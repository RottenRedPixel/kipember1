# Ember Review Notes Template

Last updated: March 21, 2026

Use this as a starting point in App Store Connect under App Review Information.

## Review notes

Ember is a memory-preservation app that lets users upload a photo or video, invite contributors, and build a shared archive around that memory through text and voice responses.

The iPhone app is a native shell for the hosted Ember experience. Core flows available to review:

- create an account or sign in
- upload a memory
- invite contributors
- view generated memory pages
- use text and voice contribution flows

There are no in-app purchases in the current iPhone shell.
